
# Portfolio Website (dylans.site)

Trying to learn Vuejs by rebuilding my portfolio website.



## Tech Stack

**Client:** Vuejs, Vite

**Server:** Node, Netlify


## Installation

Project Setup
```sh
npm install
```

Compile and Hot-Reload for Development
```sh
npm run dev
```

Compile and Minify for Production
```sh
npm run build
```
